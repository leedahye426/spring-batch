# 스프링 배치 실습

- 스프링 배치의 핵심은 <span style="color:red">기록</span>
- 하나의 job은 여러 개의 step으로 구성
- 특정 job을 같은 인자로 입력하여 시도하면, 재시도로 인식
- 재시도의 경우 기존에 성공이력이 있다면 다시 수행하지 않고, 스킵했다는 기록만 추가
